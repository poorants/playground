# Basic

## Basic grammar

```css
selector {
  property: value;
}
```

## CSS link method

```html
<head>
  <meta chaset="UTF-8" />
  <title>css link</title>

  <!-- external style -->
  <link rel="stylesheet" href="style.css" />

  <!-- interior style -->
  <style type="text/css">
    body {
      color: navy;
      font-size: 14px;
    }
  </style>
</head>
```
