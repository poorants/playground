# CSS table style

## border-collapse
표 테두리를 병합할지 또는 통합할지 여부를 표시. 셀과 셀 사이를 하나의 벽으로 할건지 두개의 벽으로 나눌건지.

|  value   | content | default |
| :------: | :-----: | :-----: |
| separate |         |    v    |
| collapse |         |         |

## table-layout
표의 너비와 일치하도록 셀의 너비를 균일하게 표시 

| value | content | default |
| :---: | :-----: | :-----: |
| auto  |         |    v    |
| fixed |         |         |


## caption-side
캡션이 있는 테이블 제목 표시 

| value  | content | default |
| :----: | :-----: | :-----: |
|  top   |         |    v    |
| bottom |         |         |


## vertical-align
표의 높이와 일치하도록 셀의 너비를 균일하게 표시 

| value  | content | default |
| :----: | :-----: | :-----: |
| middle |         |    v    |
|  top   |         |         |
| bottom |         |         |
