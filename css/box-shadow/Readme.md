# box-shadow

박스에 그림자 효과를 넣는 옵션

|  value   |              content               |  etc  |
| :------: | :--------------------------------: | :---: |
|  inset   |            Inner shadow            |       |
| offset-x | The horizontal shadow offset value |       |
| offset-y |    Vertical shadow offset value    |       |
|   blur   |          blurring effect           |       |
|  spread  |            Shadow size             |       |
|  color   |            Shadow color            |       |
