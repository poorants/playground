# box-model

속성 : width, height, margin, padding, border, outline

## border-style

| value  |       content        |
| :----: | :------------------: |
| solid  |         실선         |
| dashed |       긴 점선        |
| dotted |         점선         |
| double |        2라인         |
| groove | 안쪽으로 들어간 라인 |
| inset  |  안쪽 그림자 스타일  |
| outset | 바깥쪽 그림자 스타일 |

# outline

- border를 감싸고 있는 라인
- border와 margin 사이에 위치
- border 안쪽에 위치하려면 '-' 값
